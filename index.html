BOX-MODEL

Box model is the most fundamental topic in css, undertanding box-model will guide you to you make the best decision in designing your web pages. Everything displayed by css is a box. the following podcast explains in detail the box model: Box-Model

The maroon color which is the innermost box is the content box. The content box contains the content of the element. it could be a text, a picture or what ever you decide to put in it.

The padding box is the first box outside the content box.The padding box is used to create spaces around the content.We will see this in detail later.

The border box is outside the padding box. The border box is like the frame that surrounds pcitures.

The margin box it the outermost box in the mox model. The margin box creates spaces between element. The margin box does not affect the size of an elemnt but the other boxes do.
